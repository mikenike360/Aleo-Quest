## For Developers

Want to build with Aleo? Here's where to start:

### Quick Start
1. **[Leo Documentation](https://developer.aleo.org)** — Complete programming guide
2. **[Leo Playground](https://play.leo-lang.org/)** — Write code in your browser (no install)
3. **[Code Examples](https://github.com/AleoHQ/leo/tree/mainnet/examples)** — Real programs to learn from

### What You Can Build
- Private DeFi (DEXs, lending, derivatives)
- Confidential voting systems
- Privacy-preserving healthcare apps
- Anonymous credential systems
- Private gaming and NFTs

### Install Leo Locally
```bash
# Install the Leo CLI
curl -L https://raw.githubusercontent.com/AleoHQ/leo/mainnet/leo.sh | sh

# Create a new project
leo new my_app

# You're ready to code!
```
